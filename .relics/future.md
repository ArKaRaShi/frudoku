# Future Features

## Customization

- [ ] **Custom Fruits** - User can pick their own 9 emojis/fruits
- [ ] **Theme Selection** - Pre-built fruit themes (tropical, berries, citrus)
- [ ] **Number Mode** - Toggle between fruits and classic numbers

## Gameplay

- [ ] **Notes/Pencil Marks** - Add small notes in cells
- [ ] **Undo/Redo** - Move history
- [ ] **Hint System** - Reveal one correct cell
- [ ] **Mistake Limit** - Optional 3-strike mode
- [ ] **Auto-Check** - Highlight conflicts in real-time

## Progress

- [ ] **Local Storage** - Save current game progress
- [ ] **Statistics** - Games played, win rate, best times
- [ ] **Daily Challenge** - One puzzle per day, same for all users
- [ ] **Achievements** - Milestones and badges

## Multiplayer

- [ ] **Real-time Race** - Two players, same puzzle, who finishes first
- [ ] **Leaderboards** - Global rankings by time

## Accessibility

- [ ] **Keyboard Navigation** - Arrow keys to move, number keys to input
- [ ] **High Contrast Mode** - Better visual distinction
- [ ] **Screen Reader Support** - Aria labels for grid cells
